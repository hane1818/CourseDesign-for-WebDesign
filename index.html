Web Architecture

Front end v.s Back end

HTML

HTML基本架構

<!DOCTYPE html>
<html>
    <head>
          //單標籤
          //前後包夾標籤
    </head>
    <body>
        //網頁內容顯示
    </body>
</html>

其他常用標籤：換行(br)、標題(h1~h6)、段落(p)、清單(ol, ul)、連結(a)、圖片(img)

兩種標籤元素

  • 行內元素 : <span>, <a>
  • 區塊元素 : <div>, <p>

CSS

三種使用方式

  • 行內樣式  
  • 置頂樣式   
    <head>
    
</head>
  • 外部樣式 
    <head>
    
</head>

三種基本選擇器(selector)

  • Tag selector 
    body {
    background: red;
}
  • Class selector : 標籤中可有多個類別，且可重複使用 
    .container {
    background: blue;
}
  • ID selector : 每個標籤只能有一個，且名稱不可重複 
    #main {
    background: yellow;
}
